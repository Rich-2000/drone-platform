ℹ️ Upload Info
01Images are stored on Cloudinary CDN under naimos/drones.
02Uploads use a server-signed URL — your Cloudinary secret is never exposed to the browser.
03The NAIMOS HQ platform at naimos.vercel.app fetches images from this platform's /api/latest endpoint in real time.
04Metadata (mission, region, GPS, drone ID) is stored as Cloudinary context and in IndexedDB for offline access.
🔗 Programmatic Upload API
POST directly to this platform from ground station software or custom scripts:
POST /api/upload
Content-Type: multipart/form-data

Fields:
image: <file>
mission: "NAIMOS-M-2026-044"
region: "Western Region"
droneId: "NAIMOS-UAV-01"
type: "rgb" | "thermal" | "ndvi"
lat: 5.8456 lon: -1.8234 alt: 120
notes: "Active pit at GPS coords"
